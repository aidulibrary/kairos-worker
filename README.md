# KAIROS — 几

> The Kairos Moment — 原生AI市集社交电商平台

四种身份，同一个Kairos。

- **创造者**：清出场域，召唤Kairos降临
- **到来者**：带着手艺和故事赴约
- **降临者**：走进来的那一刻，Kairos完成
- **助成者**：手艺是Kairos的骨骼

## 快速开始

```bash
npm install
cp .env.example .env.local  # 填入 DeepSeek API Key
npx prisma generate
npm run prisma:seed
npm run dev
```

访问 http://localhost:3000
